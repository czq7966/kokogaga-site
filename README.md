Adhoc Cast Signaler-Server
===
自组网WebRTC信令服务器，基于开源的socket.io上开发

#安装步骤
1. npm install adhoc-cast-signaler-server
1. cd ./node_modules/adhoc-cast-signaler-server
1. npm install
1. npm start


#Signal-Center Dev启动流程
1. npm run server:dev
1. npm run amd:dev
